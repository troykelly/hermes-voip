# docs/ — canonical documentation

Source of truth for every number and policy this project presents. When a figure here and a
figure elsewhere disagree, this directory wins — fix the other place. Cite sections (e.g.
"stack.md §2") when encoding a figure into code. Strategy docs are operator-owned: propose
edits via commits, but flag substantive policy/price/legal changes to the operator rather
than silently rewriting them. Subtrees: `adr/` (decisions), `runbooks/` (operations).
