# docs/adr — architecture decision records

Numbered `NNNN-kebab-title.md`, zero-padded to four digits; `0000-template.md` is the
template. Use the `adr` skill to add one. Records are `Accepted` when written (we record
decisions, not proposals); a reversal edits the old record's status to
`Superseded by ADR-NNNN` in the same commit that adds the superseding record. ADRs are the
WHY; operational HOW lives in `docs/runbooks/`.
